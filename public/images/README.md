# Profile Image Directory

To update your profile image on the portfolio, simply upload or place your photograph in this directory and name it:

`profile.jpg`

The website is pre-configured to load `/images/profile.jpg` automatically as your primary portfolio portrait. If no image is present here, a beautiful, high-tech animated placeholder is displayed by default.

## Specifications
- **Recommended Ratio:** 1:1 (Square) or 4:5 portrait crop
- **Recommended File Format:** JPG or PNG
- **Recommended Dimensions:** 800px x 800px
